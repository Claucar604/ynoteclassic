Ynote Classic

Ynote Classic is an easy to use, open and versatile Text and Source Code Editor. It can deal with a variety of tasks ranging from simple writing to complex programming. It makes editing a more easier and comfortable task with it's powerful editing operations, Commands and shortcuts.Syntax highlighting, code folding, indentation, snippets, macros and plugins add more to it's functionality

http://ynoteclassic.codeplex.com

Features

  • Multi-Level Undo/Redo
  • Syntax Highlighting, Code Folding, Auto-Indent, Auto-Completion
  • Powerful Search and Replace using Regular Expressions in document, files and selection.
  • Powerful Project Management
  • Diff Utility
  • Automatic Encoding Detection and support for all codepages.
  • WYSWYIG Printing
  • MDI and SDI
  • Extensible with Plugins, Macros, Color Schemes, Syntax Files, Scripts and much more.
